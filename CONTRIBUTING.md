# Contributing to Yaaram / Couple Memories

Thanks for helping improve this open couple memories app.

## Development
1. Fork and clone the repo
2. Copy `.env.example` to `.env` and fill Cloudinary values
3. Follow `FIREBASE_SETUP.md`
4. Run `flutter pub get` then `flutter run`

## Pull requests
- Prefer small focused PRs
- Include a short test plan
- Do not commit `.env` or private keys

## Code style
Match existing Flutter/GetX patterns and the light theme shells.
